Discord sucks

Discord sucks. Not only because it's proprietary software, its the the worst kind of proprietary. They actively pursue not only companies, but users who committed the evil act of.. uh.. inserting CSS to make a chat client looks better?
Let me explain. BetterDiscord is a free-software modification for the discord client which, to put it simply, inserts custom CSS into the client, which simply changes the U.I. Nothing less, nothing more. Now, let me explain further. Discord actively looks for and bans users who are caught using the modification. Why? why is enforcement of this necessary?
Another thing discord bans users for, is using free-software clients. Personally I don't understand that. If I was a big software proprietor, I would let people use those because either way it would make me money. The same data is being collected, stored, and then later sold to advertisers. It would not make any difference in profits. And even if it did, a little less money is better than no money?? I really don't understand.
This really is an example of how proprietary software, and Copyright laws in general, does not breed innovation, but rather discourage it.
